# 💼 Igor Luiz — Soluções Digitais

Site institucional desenvolvido para apresentar meus serviços como desenvolvedor web, com foco em design moderno, performance e experiência do usuário.

---

## 🚀 Sobre o projeto

Este projeto representa minha presença digital como desenvolvedor, com o objetivo de demonstrar habilidades técnicas e capacidade de criar interfaces profissionais.

O site inclui:

- Seção de apresentação (Hero)
- Área "Sobre"
- Serviços oferecidos
- Projetos fictícios para demonstração
- Formulário de contato
- Layout totalmente responsivo

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

## 💡 Objetivo

Criar um site profissional capaz de:

- Apresentar meus serviços de forma clara
- Demonstrar domínio em front-end
- Servir como base para futuros projetos reais
- Evoluir com novas funcionalidades
